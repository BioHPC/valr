Generate randomly placed intervals on a genome.

Usage

bed_random(genome, length = 1000, n = 1e+06, seed = 0)

Arguments

genome
genome tbl
length
length of intervals
n
number of intervals to generate
seed
seed RNG for reproducible intervals

Description

Generate randomly placed intervals on a genome.

Examples

genome <- tibble::tribble(
  ~chrom,  ~size,
  "chr1",  10000000,
  "chr2",  50000000,
  "chr3",  60000000,
  "chrX",  5000000
)

# random intervals (unsorted)
bed_random(genome)
