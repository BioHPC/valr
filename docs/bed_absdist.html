Compute the absolute distance between query intervals and reference intervals

Usage

bed_absdist(x, y, genome)

Arguments

x
tbl of intervals
y
tbl of intervals
genome
genome tbl

Value

data_frame

Description

Compute the absolute distance between query intervals and reference intervals

Examples

x <- tibble::frame_data(
~chrom,   ~start,    ~end,
"chr1",    75,       125
  )

y <- tibble::frame_data(
  ~chrom,   ~start,    ~end,
  "chr1",    50,       100,
  "chr1",    100,       150
  )

genome <- tibble::frame_data(
  ~chrom, ~size,
  "chr1", 500,
  "chr2", 1000
)

bed_absdist(x, y, genome)
